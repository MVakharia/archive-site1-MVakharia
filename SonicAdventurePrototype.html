Sonic Adventure - Character Controller Demo

A demo developed from September 2017 to January 2018, built using Unity and C#, with assets created in Blender and GIMP.

The demo showcases my attempt at recreating a character controller from the console game Sonic Adventure 2, which was originally released for the Sega Dreamcast in 2001.

Best experienced with a controller.
